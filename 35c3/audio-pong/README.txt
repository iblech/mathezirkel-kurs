volume-costreaming.py macht auf localhost:8080 einen Webserver auf.
Dort /volume-costreaming.html aufrufen. Damit wird via JavaScript die
Lautstärke geholt und in eine Datei unterhalb von /tmp geschrieben.

Das eigentliche Spiel ist dann audio-game-costreaming.py. Lautstärkeminimal-
und -maximalwerte oben bei den Konstanten geeignet anpassen!
